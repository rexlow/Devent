#DEVENT

##User Manual
**Devent** is an event app powered by **React Native**.

It is supported by both **iOS** and **Android**.

##System Requirement
OS X 10.11 and above

Xcode 7.0 and above

node >= 4.1

##Get Started
Unzip Devent.zip

~~~
cd Devent
npm install
npm install rnpm -g
rnpm link
react-native link
react-native run-ios or react-native run-android
~~~
